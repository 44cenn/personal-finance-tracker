# Personal Finance Tracker

Aplikasi web untuk mencatat dan menganalisis pemasukan serta pengeluaran pribadi.

## Teknologi

- PHP
- MySQL
- Bootstrap 5
- JavaScript
- XAMPP

## Fitur

- Register, login, dan logout
- Role Admin dan User
- CRUD pemasukan
- CRUD pengeluaran
- CRUD kategori
- Riwayat transaksi
- Filter dan pagination
- Pengaturan tema, mata uang, dan bahasa
- Upload foto profil
- Dashboard ringkasan keuangan

## Cara Menjalankan Project

1. Download atau clone repository ini.
2. Pindahkan folder project ke:

   C:\xampp\htdocs\personal-finance-tracker

3. Jalankan Apache dan MySQL melalui XAMPP.
4. Buka phpMyAdmin.
5. Buat database bernama:

   personal_finance_tracker

6. Import file:

   database/personal_finance_tracker.sql

7. Buka:

   http://localhost/personal-finance-tracker/
